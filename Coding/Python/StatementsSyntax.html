1 对齐即为同一级

except与else对齐，是一级的。 

while True:
    reply = input('Enter text: ')
    if reply == 'stop':   break
    try:
        int(reply)
    except:
        print('Bad ' * 8)
    else:
        print(int(reply) ** 2)
print('Bye!')

4 extended-sequence-unpacking

*是个标识，标志着下个variable是个list，并收集所有未被赋值的所有其它变量 

a, *b = 'spam'
==>a = 's', b = ['p', 'a', 'm']

>>> a, b = 'spam'
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: too many values to unpack (expected 2)

*可以出现在任何地方，总是先赋完其它亦是再最后赋 * 

>>> x = [1, 2, 3, 4]
>>> *a, b = x
>>> b
4
>>> a
[1, 2, 3]
>>> a, *b, c = x
>>> a
1
>>> b
[2, 3]
>>> c
4

与slicing的区别: a sequence unpacking alignment return a list for multipe matched items. while slicing returns a sequence of the same type as the object model. 

>>> x = '01234'
>>> x[0], x[1:4], x[4]
('0', '123', '4')         #x[1:4]类型与'01234'相同
>>> a, *b, c = x
>>> a
'0'
>>> b                     # b为list类型
['1', '2', '3']
>>> c
'4'
