* options
** -E
   显示预处理完成后（编译前）的代码。（展开宏定义等）
   例如要查看ssize_t具体是什么，可以 gcc -E test.c生成预处理后代码查看。

** @filename
   可以把选项放到文件中，GCC会将文件中的内容展开加到选项中。比如:filename中内容为：-o test test.c，此时执行:gcc @filename等价于执行gcc -o test test.c
