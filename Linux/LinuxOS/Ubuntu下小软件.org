* 温馨提示notify-send
notify-send是个实用的小工具，可以在桌面的右上角向用户发送提醒，此工具再配合 crontab 命令便可实现定时向用户发送提醒。
安装：sudo apt-get install libnotify-bin
使用：notify-send -i icon目录 -t 提示显示的时长 "提醒内容"
可以写在脚本中再配合 crontab 命令来定时发送提醒。
