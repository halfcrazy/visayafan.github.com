#+title:德语中的后缀
参考 [[http://en.wiktionary.org/wiki/Category:German_suffixes][Wiktionary-Category:German suffixes]]
* -ig
  形容词后缀，名词/动词词干/副词+ig
  | sand 沙子        | sandig 含沙的        |
  | richt v对准/校准 | richtig =right正确的 |
  | schlaf der睡觉 | schläfig 困的      |
* -haft
  形容词后缀表：像...样的，...似的
  | sagen 说 |sagenhaft 传说的/惊人的/传奇的|
* -los
  表without
  | arbeits 前缀表工作相关的 |arbeitslos 失业的|
* -arm
  adj后缀，表缺少的
  | fantasie die想象力/幻想 | fantasiearm 缺乏想象力 |
  | sorge die关心/担心      | sorgearm 不担心的          |
  | regen der雨             | regenarm 无雨的        |
* -frei
  表 free from .../non-...
    | arbeits | arbeitsfrei =free from work 没事干 |
* -leer
  without
  | luft =air | airleer =airless真空 |
* -reich
  reich =rich
  后缀时表多的
  | vitamin | vitaminreich |
* -fest
  fest =stable
  后缀表稳固的
  | wasser | wasserfest =watertight防水的 |
* -isch
  可以用来表示国家/民族的
  | China           | chinesisch |
  | Russland 俄罗斯 | russisch   |
  | Latein 拉丁     | lateinisch |
* -chen
  表小的
  | Hund der狗 | Hündchen 小狗  |
  | Lamp 灯    | Lämpchen 小灯   |
  | Mädel 女孩 | Mädchen 小女孩  |
  | Mann       | Männchen 小男人 |
  | Kind       | Kindchen        |
* -lein
  表小的
  | Frau  | Fräulein 少女  |
  | Buch  | Büchlein 小书 |
  | Vogel | Vöglein 小鸟   |
  | Haus  | Häuslein       |
  | Tisch | Tischlein      |
  | Kind  | Kindlein       |
  | Mann  | Männlein       |
* -schaft
  相当于-ship/-hood
  | Freund  | Freundschaft           |
  | Mann    | Mannschaft 全体成员/组 |
  | Partner | Partnerschaft 伙伴关系 |
  | Wissen  | Wissenschaft           |
  | Wirt    | Wirtschaft             |

* -lich
  相当于英语中的-ly
  | person | persönlich 个人的          |
  | Kind   | Kindlich 儿童的            |
  | gut    | gütlich 友好的/心平气和的 |
  | Frau   | fraulich 女性的            |
  | Mann   | männlich 男性的            |
  | Freund | freundlich 友好的          |
* -heit
  将形容词转换成名词，相当于英语中的-ness，-hood
  | Kind              | Kindheit =childhood |
  | schön             | Schönheit =beauty   |
  | Frei              | Freiheit 特权       |
  | Ein 不定冠词 一个 | Einheit 一个单位/单元     |
* -keit
  与heit后缀意思一样，但用在以 -ig, -bar, -lich, -sam结尾的词后面。
  | langsam           | Langsamkeit =slowness |
  | persönlich 个人的 | persönlichkeit 个性 |
  | wirklich 实际上   | Wirklichkeit die现实  |
