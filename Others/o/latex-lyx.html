4 公式

4.1 自定义TAG

可以用\tag命令来自定义TAG，但在lyx中要先将公式环境切换到ASM环境：Edit->Math->Chang Formula Type->ASM align environment。

4.2 长公式换行

参考
在公式上右键 AMS enviroment->AMS multiline enviroment，开始输入公式，此公式是左对齐的，在需要换行的地方Ctrl-RET即可新建一行此行的公式是上一行公式的继续，是右对齐的。
注意：用 Alt-m ( 类似命令生成的括号内的内容被lyx看成整体不可分割，所以不能在这种括号里面换行（实在没有办法只有用手工输入括号的办法了，这样输出结果括号会略小）。

5 插入源码

  1. 可以通过 insert->program listing 来插入源码，不过此时源码是没有高亮的。
  2. 可以通过 insert->file->child document来选择源码所在的文件，include type选择program listing，可选标题，more parameters中可以输入language=C++可以通过？来查看更多可用语言。

默认源码标题类似 Listing 1:标题，可以通过 \renewcommand{\lstlistingname}{源码} 变成 源码1：标题

10 源码高亮包

lstlisting可以实现源码高亮但颜色单调

\begin{lstlisting}[language=matlab]
test
\end{lstlisting}

可替代之的是minted。
具体参考：http://www.latexstudio.net/?p=596

\documentclass{article}
\usepackage{minted}
\begin{document}
\begin{minted}[linenos=true]{c++}
#include<iostream>
using namespace std;

class A{};
class B{};

class C{
public:
     operator A(){}
     operator B(){}
};

void f(A a){}
void f(B b){}

int main(int argc, char *argv[])
{
     C c;
     f(c);
     return 0;
}
\end{minted}
\end{document}

12 Beamer中为插入frame设置快捷键

Tools->Preferences->Editing->Shortcuts中New对话框中Function栏写 layout BeginFrame 并为其添加快捷键，同理可为 layout EndFrame 添加快捷键.
查看一个功能对应的函数：在菜单上执行一次此功能后会出现在状态栏上。
默认文件保存到了~/.lyx/bind/user.bind，可以备份此文件以便日后使用。

13 lyx解决Beamer中插入代码问题

直接insert->program listing插入代码这种方法行不通( *用这种方法插入代码可以在边框上右键setting来进行lstset设置)
目前最好的解决办法是把待插入的代码保存到一个文件中，通过 insert->file->child document来插入文件，在include type下选择program listing，此时可在More Parameters中设置相应的参数：

  1. basicstyle={\tiny} 设置字体 \tiny \scriptsize \footnotesize \small \normalsize \large \Large \LARGE \huge \Huge
  2. breaklines=true 适当时候断行显示
  3. commentstyle={\color{black}} 注释颜色为黑色（否则如果不设置中文为黑色，英文为红色）
  4. language=Matlab 设置代码所用的语言
  5. numbers=left 显示行号，可选在显示左边还是右边
  6. numberstyle={\tiny\color{lightgray}} 设置行号大小及颜色
  7. frame=shadowbox 显示边框，可选 leftline, topline等
  8. rulesepcolor={\color{red!20!green!20!blue!20!}} 若边框为shadow则设置阴影的颜色

模板：

basicstyle={\tiny}
breaklines=true
commentstyle={\color{black}}
language=Matlab
numbers=left
numberstyle={\tiny\color{lightgray}}


注意： 如果注释中存在中英文混排情况则夹在中文之间的英文前后必须有空格，否则排版出错.

14 lyx的Beamer Overlay控制

14.1 columns

首先从下拉菜单下选择columns，在新行中选择column，输入宽度，后快捷键 M-S-right 将其提升了columns的child。

14.2 block

菜单下选择block后 C-l 输入 {标题} ，注意不可省略 {} 。

14.3 itemize/enumerate

M-p e 创建enumerate，可以在行首 C-l 输入：

  1. <n1-n2>内容 表示在n1至n2个幻灯片上显示
  2. <only@n> 表示只在第n个幻灯片上显示








$$\underrightarrow{\ \ result\ \ }$$






右图中列表是动态出现的。

14.4 overprint

如果一部分内容在某些不连续的幻灯片上出现可以用overprint，然后添加tex代码 \onslide<n1,n2,..> n1,n2……表示在哪些幻灯片上显示

14.5 alt

\alt<n>{before}{after} n帧前为before，n帧后为after。

14.6 temporal

\temporal<n>{before}{on}{after} 在n帧之前一直是before，n帧时是on，之后是after

14.7 创建链接

创建target: \hypertarget{targetname}{text-to-show} 。
创建link: \hyperlink{targetname}{\beamergotobutton{text-to-show-on-button}}。
其它有用的button： \beamerreturnbutton，\beamerbutton

14.8 整个frame的内容动态出现

可以在菜单选择BeginFrame后 C-l 输入 <+-> 再输入标题。这样如果此frame此都是列表则列表逐一出现。

15 打印Beamer制作的幻灯片

15.1 修改源码实现

Beamer做的幻灯片的pause之类的Overlay都是通过额外的PDF页来实现的，打印时会造成浪费，解决方法是：

\documentclass[handout]{beamer}

这样生成的PDF不再具有Overlay的特效，但适合打印。(前提是你有此Beamer制作幻灯片的源码)
lyx设置：document->setting->document class->class options->custom中添加handout

15.2 利用pdfpages包实现

大部分情况下我们手上只有一份pdf版的幻灯片，没有源码，此时上面的方法就行不通了，可以利用如下方法实现：

\documentclass{article} %
\usepackage{pdfpages}
\begin{document}
\includepdf[nup=2x2,landscape,delta=5mm 5mm, scale=0.95,pages={1-18}]{trans.pdf}
\end{document}

其中1-18指定了想要合并trans.pdf的页码范围，2x2指定了每页上个数。
